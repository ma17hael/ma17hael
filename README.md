# ma17hael

Bienvenue sur le dépôt GitHub de **ma17hael**, développeur freelance spécialisé dans le développement web et la création de jeux.

## À propos de moi

Je suis un développeur freelance passionné par la conception de solutions logicielles variées : du développement web moderne (HTML, CSS, PHP, JavaScript) à la programmation C++ orientée jeux et systèmes interactifs. Mes projets combinent une approche structurée et respectueuse des standards tout en intégrant des éléments créatifs et ludiques.

## Compétences

- **Langages :** PHP, JavaScript, HTML5, CSS3, C++, SQL, JSON  
- **Frameworks et outils :** SDL2, WordPress, Tailwind CSS, Figma  
- **Bases de données :** MySQL, PDO  
- **Projets :**  
  - Jeux RPG et tycoon avec système d’inventaire, ressources et gestion de personnages  
  - Sites vitrines et SPA avec design responsive et animations interactives  
  - Intégration de données externes (Google Places, JSON, fichiers KMZ)
